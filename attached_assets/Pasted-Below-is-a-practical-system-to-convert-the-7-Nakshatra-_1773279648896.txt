# 1. Core Nakshatra Classification (7 Types)

In Vedic astrology the zodiac is divided into **27 segments of 13°20′ each**, called Nakshatras, representing the Moon’s position in the sky. ([Wikipedia][1])

These are grouped into **seven categories based on their energetic nature**. ([Drik Panchang][2])

### The Seven Groups

| Category  | Sanskrit Name    | Nature     | Typical Use         |
| --------- | ---------------- | ---------- | ------------------- |
| Fixed     | Dhruva / Sthira  | Stable     | long-term work      |
| Movable   | Chara            | Dynamic    | travel & movement   |
| Cruel     | Ugra / Krura     | Aggressive | destructive acts    |
| Ordinary  | Mishra           | Neutral    | mixed works         |
| Short     | Kshipra / Laghu  | Quick      | fast actions        |
| Gentle    | Mridu            | Soft       | relationships & art |
| Ferocious | Tikshna / Daruna | Intense    | transformation      |

# 2. Nakshatra → Category Mapping

### Fixed (Dhruva)

Stable and long-lasting actions.

* Rohini
* Uttara Phalguni
* Uttara Ashadha
* Uttara Bhadrapada

Best for:

* Buying property
* Laying foundations
* Agriculture
* Starting long-term business
* Marriage alliances
* Government positions

Avoid:

* Temporary work
* Short-term trades
* Travel

---

### Movable (Chara)

Good for motion and change.

Nakshatras:

* Punarvasu
* Swati
* Shravana
* Dhanishtha
* Shatabhisha

Best for:

* Travel
* Transport
* Business trade
* Changing jobs
* Moving house
* Negotiations

Avoid:

* Foundations
* Real estate purchases
* Permanent commitments

---

### Cruel (Ugra)

Aggressive and forceful energy.

Nakshatras:

* Bharani

* Purva Ashadha
* Purva Bhadrapada

Best for:

* Surgery
* Military acts

* Fire or chemical work
* Competitive actions

Avoid:

* Marriage
* Spiritual ceremonies
* Diplomacy

# 3. ICS Calendar Design

Google Calendar events should represent **the Moon entering a Nakshatra**.

Each Nakshatra lasts roughly **~1 day**.

Calendar event fields:

### Title

```
Nakshatra: Rohini (Fixed / Dhruva)
```

### Event Length

From:

```
Moon enters Rohini
```

To:

```
Moon enters next Nakshatra
```

You must pull these times from:

* Drik Panchang API
* Swiss Ephemeris
* Astro-Seek export
* Panchang API
