# **UI DESIGN EXECUTION MANUAL FOR THE REPLIT AI AGENT**

## **Purpose**

Your task is to transform the existing UI of the “Ibn Arabi’s Cosmology” app into a modern, premium-quality interface comparable to Airbnb, Calm, or Headspace. You **must enforce design consistency**, clear hierarchy, and component-based UI patterns.

Every output you generate must follow the rules below.

---

## **GLOBAL DESIGN RULES**

### 1) **Layout Grid**

* Max container width: **1120px**
* Horizontal padding: **24px** on desktop, **16px** on mobile
* Vertical spacing between sections: **48–64px**
* Base spacing unit: **8px**
* All UI elements must align to this grid. No free-floating components.

### 2) **Typography**

Use a **two-font system**:

**Primary UI Font (Latin):** `Inter` or `SF Pro`
**Arabic Font:** `Cairo` or `IBM Plex Arabic`

**Scale**

* App Title (H1): 42–48px, bold
* Section Title (H2): 28–32px, semibold
* Card Title (H3): 22–24px, semibold
* Body Text: 16–18px, regular
* Meta Text: 12–14px, medium

**Rules**

* Titles always dominate over timestamps.
* English is left-aligned, Arabic is right-aligned.
* Never mix more than two font families.
* Time and metadata are always in smaller weights.

### 3) **Color System**

Use a controlled palette:

**Background:** `#FAFAF6` (warm neutral)
**Card Background:** `#FFFFFF`
**Primary Text:** `#1B1B1F`
**Secondary Text:** `#6C6F7A`
**Border / Divider:** `#E5E7EB`

**Accents (planet-based):**

* Saturn: Indigo `#4338CA`
* Moon: Silver `#9CA3AF`
* Sun: Gold `#D97706`
* Mars: Red `#B91C1C`
* Venus: Emerald `#059669`

**Status colors**

* Good: `#10B981`
* Neutral: `#9CA3AF`
* Worse: `#EF4444`

You must not invent new colors. All UI must derive from these.

### 4) **Card Architecture**

Every card must follow this exact structure:

```
[Card Container]
- Rounded corners: 20px
- Padding inside: 32px
- Soft shadow: rgba(0,0,0,0.05) at 10–30px blur
- Section labels = uppercase, small, muted
- Section dividers = subtle lines with 20% opacity
```

Cards must never be borderless, flat, or visually identical to the background.

---

## **COMPONENT SPECIFICATIONS**

### **A) Hero Section**

Purpose: establish context, identity, and date

Required elements in order:

1. App title (largest element on page)
2. Subtitle (small, muted)
3. Current time, Islamic date, location selector
4. Mode toggle (Tropical/Sidereal)
5. Divider line

Time must never exceed the title in visual dominance.

### **B) Lunar Mansion Card**

Every Lunar Mansion card must include the following blocks:

```
[Card Header]
- Label: CURRENT LUNAR MANSION
- Mansion number badge
- Mansion name (EN)
- Mansion name (AR)

[Progress Bar]
- Shows elapsed percentage
- Next mansion + timestamp

[Sections - always in this order]
1. Celestial Sphere
2. Meaning
3. Best Use / Energetic Influence
4. Divine Name
5. Rituals
```

Arabic labels must be right-aligned with reduced opacity (70–80%).

### **C) Planetary Hour Card**

Always structured like this:

```
Planet Glyph (oversized, centered)
Hour Title (EN + AR)
Start – End Time
Planet association badge (colored)
Elapsed progress bar
```

Planet accent color must define the visual identity of the card.

### **D) Filters and Controls**

* Use **pills** instead of dropdowns whenever there are two or three options
* Active pill = accent background + white text
* Inactive = transparent background + accent outline

### **E) Interactions**

* Hover state: subtle elevation + shadow intensification
* Click state: slight downward transform
* Updating elements (time/progress): soft pulse animation only, no flashing
